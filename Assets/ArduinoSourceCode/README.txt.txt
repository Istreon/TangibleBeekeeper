This folder contains two arduino codes, for the tangible smoker.
One is for the use of entreprise WIFI (like eduroam)
The other one if for classical WIFI.

!!!Before upload source code into the tangible smoker, you must update login/mdp in the source code file.

You can get tangible smoker IP adress with arduino software, using monitor serial. Use this IP adress on UNITY to connect the tangible smoker to unity. 
The associated script is "ConnectedSmoker.cs".
